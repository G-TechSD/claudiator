Powerful Features

Everything you need to manage multiple Claude Code sessions

Multi-Terminal Dashboard

Run up to 16 Claude Code instances simultaneously in a flexible 1-4 column grid layout.

Session Persistence

Each terminal runs in its own tmux session, surviving browser refresh, pop-outs, and reconnections.

Pop-Out Windows

Pop terminals to separate windows that reconnect to the same tmux session seamlessly.

Path Autocomplete

Recent paths are saved (up to 25) and suggested as you type. Press Enter to create terminal.

Bypass Permissions

Configure permission bypass as the default for all new terminals for faster workflows.

Terminal Groups

Organize terminals into color-coded groups for better project management.

Quick Installation

Get started in seconds with our one-liner installer

One-Liner (Recommended) 

curl -fsSL https://cdn.jsdelivr.net/gh/G-TechSD/claudiator@main/install.sh | bash

Installs Claudiator, checks dependencies, and sets up the claudiator command.

Manual Install 

git clone https://github.com/G-TechSD/claudiator.git
cd claudiator
npm install && npm run build
npm start

Clone the repo, install dependencies, build, and start.

Docker 

docker run -d --name claudiator \
  -p 3200:3200 \
  -v ~/projects:/projects \
  gtechsd/claudiator:latest

Run in Docker with your projects mounted.
